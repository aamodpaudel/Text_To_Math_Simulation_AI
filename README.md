# Text_To_Math_Simulation_AI

Text_To_Math_Simulation_AI is an intelligent application powered by AI that bridges the gap between natural language text and mathematical simulations. It functions as an interactive math tutor agent, helping users seamlessly visualize and learn mathematical concepts.

## Features

- **Math Tutor Agent**: An AI-powered tutor built with advanced generative models to interpret and explain complex math problems.
- **Interactive Visualization**: Render 3D geometric shapes and interactive math functions using `@react-three/fiber` and `mafs`.
- **Modern Tech Stack**: Built with Next.js 16, React 19, and styled with Tailwind CSS for an optimal and responsive experience.
- **AI Integrations**: Leverages Google Generative AI and LangChain for cutting-edge text-to-math capabilities.

## Getting Started

First, install the dependencies if you haven't already:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the application in action.

## Deployment

The easiest way to deploy this Next.js app is to use the [Vercel Platform](https://vercel.com/new).
